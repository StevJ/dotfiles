# dotfiles

## Installing

Run `stow` to symlink everything or just select what you want

```sh
stow */ # Everything (the '/' ignores the README)
```

```sh
stow zsh # Just my zsh config
```

## Remember npm packages

```sh
npm -g list > npm-global.txt
```
